Mini-site cotes PMU

Ce site ne sert qu'a heberger une seule fonction serverless (netlify/functions/pmu-cotes.js) utilisee par TurfAnalyse-Web (heberge separement sur GitHub Pages) pour recuperer automatiquement les cotes PMU sans blocage CORS.

Aucune page n'est destinee a etre consultee ici. Si vous voyez ce message, le deploiement fonctionne correctement.

Endpoint disponible :
/.netlify/functions/pmu-cotes?date=DDMMYYYY&reunion=1&course=4&type=participants
